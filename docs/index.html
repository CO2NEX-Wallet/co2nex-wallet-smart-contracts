🔐 Escrow Flow — How It Works

  1. Buyer: Calls createDeal() passing the seller, token ID, and amount.
  2. Escrow Holds Tokens: Tokens are securely held until released.
  3. Release Deal: Buyer or contract owner calls releaseDeal(dealId).

🚀 Quick Start

1️⃣ Clone Repository

git clone https://github.com/CO2NEX/co2nex-wallet-smart-contracts.git

2️⃣ Install Dependencies

npm install

3️⃣ Create .env File

cp .env.example .env 
POLYGON_RPC_URL=https://polygon-rpc.com
ETHEREUM_RPC_URL=https://mainnet.infura.io/v3/YOUR_INFURA_KEY
PRIVATE_KEY=YOUR_WALLET_PRIVATE_KEY

4️⃣ Compile Contracts

npx hardhat compile

5️⃣ Deploy Locally

npx hardhat run scripts/deploy.js --network localhost

➡️ Deploy to Polygon

npx hardhat run scripts/deploy.js --network polygon
